# Sound Placeholders

Drop your own sound files in this folder using these exact filenames:

- `ui-click.mp3`
- `boot-complete.mp3`
- `ambience-loop.mp3` (looping background ambience)
- `ability-pulse.mp3`
- `ability-emp.mp3`
- `ability-scan.mp3`
- `ability-overclock.mp3`
- `theme-switch.mp3`
- `camera-switch.mp3`
- `presentation-toggle.mp3`
- `audio-enabled.mp3`
- `audio-disabled.mp3`
- `audio-error.mp3`

## Notes

- Keep files short for UI sounds (100-600 ms) so controls stay responsive.
- `ambience-loop.mp3` should be seamless if possible.
- The app will continue working even if some files are missing.
